# Installation

## Extensions Platform Installation

1. Go to `Edit > Preferences > Get Extensions`
2. Search "SPArk Sequencer"
3. Click Install

## Manual Installation

1. Download the [Latest Release](https://github.com/NickTiny/SPArk-sequencer-addon/releases)
2. Go to `Edit > Preferences > Add-ons`.
3. Click `Install...` and select the addon ZIP or folder.
4. Enable `SPArk Sequencer Addon`.
5. Optional: `opentimelineio` for editorial import/export support, see [OpenTimelineIO installation](../CONTRIBUTING.md#installing-opentimelineio).
